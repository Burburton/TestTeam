# Amazing Team

AI-powered autonomous development team that implements projects driven by issues.

## Features

- **Automated Workflow**: Issues trigger automatic analysis, design, implementation, and review
- **AI Team**: Architect, Developer, QA, and Reviewer agents working together
- **GitHub Integration**: Seamless integration with GitHub Issues, PRs, and Actions
- **Quality Assured**: Built-in testing, linting, and code review

## Quick Start

1. Create an issue using one of the templates
2. Comment `/opencode use auto to process this issue`
3. Watch the AI team work through the implementation
4. Review and merge the resulting PR
